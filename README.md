# EA1
## Activity Description
During this event (Bigmode Game Jam), participants were tasked with creating a game related to the theme "Slick" in 9 days. My team and I created a game called ProtoPunk within that time which is a fast-paced, steampunk movement FPS. 

## Technical Decisions
Many of the technical decisions made were related to the scope of the project. Since we only had 9 days, we had to aggressively cut features while maintaining the fun aspects of the game. Some features that didn't make it were leaderboards, enemy movement and attacks, and a splatoon-like attack system where the player could "paint" surfaces using UV mappings. Otherwise, the main technical decisions were related to the player controller (movement mechanics), menus, and save data.

## Contributions
I participated in a team of 6 for this project. My contributions were connecting the main gameplay system of enemies giving you speed to the player controller, making level elements such as destructible crates, reset triggers, and other modifications to the player controller to improve game feel.

## Quality Assessment
I feel I did a good job with this event although there were times where I was working on features for a few hours that had to be scrapped eventually. If I could do this again, I would put more focus on underlying systems as a foundation and scope appropriately to be able to add leaderboards in the game jam build. We ended up getting 11th most rated game (with 92 ratings), out of 648 total entries. The results are not out yet but it will be exciting to see where we rank in terms of quality.
